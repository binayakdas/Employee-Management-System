How to run the project in personal computer :-

1) Download and install XAMPP server.

2) Open XAMPP Control panal and start [apache] and [mysql].

3) Extract the files from the zip in  C:\xampp\htdocs.

4) Open the following link in the browser.

   http://localhost/phpmyadmin/

5) click on new at side navbar.

6) give a database name as (datatable_db) hit on create button.

7) after creating database name click on import.

8) browse the file in directory[datatable/datatable_db.sql].

9) after importing successfully.

10) open any browser and type http://localhost/datatable/index.php

11) The web page will be opened.
